# Политика безопасности

## Поддерживаемые версии Bitrix

Ниже указанны версии бека на котором работает текущий код с репозитория. В основе проекта на сервере установлен BitrixVM.

|    Version    |     Supported      |
| ------------- | ------------------ |
| PHP < 7.4.x   | :x:                |
| PHP > 8.1.x   | :white_check_mark: |
| MySQL > 5.4.x | :white_check_mark: |
